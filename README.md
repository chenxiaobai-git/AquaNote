 AquaNote — 水族养护科研级软件

专为水族爱好者打造的本地数据管理桌面软件，支持多维度数据记录、智能分析与 3D 可视化。

## 功能概览

- **3D 可视化建模** — 实时等轴测线框模型，直观展示鱼缸与过滤系统空间关系
- **水质实验室** — 追踪 pH、氨氮、亚硝酸盐、硝酸盐等关键指标，生成趋势图
- **环境检测** — 智能评估水体安全边界，生成 0-99 分环境评分
- **生物档案** — 完整记录每一尾生物的来源、入缸日期与生长历程
- **维护日历** — 周期性任务设定与自动提醒
- **耗材库存** — 饲料、药品统一管理，低库存预警
- **AI 水族助手** — 内置专家知识库，支持文心大模型（免费）与 DeepSeek 切换

## 技术栈

Vite + TypeScript + React + Tailwind CSS + shadcn/ui

## 本地开发

### 环境要求

- Node.js ≥ 20
- npm ≥ 10

### 快速开始

```bash
npm i
npm run dev -- --host 127.0.0.1
```
